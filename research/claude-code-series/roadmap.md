## 交叉維度 — PC↔雲邊界

核心洞察：PC 性能過剩，雲端極度稀缺。CC 的邊界在哪？如何優化？

| 維度 | 問題 |
|------|------|
| 本地執行層 | 文件、Bash、Git、Tree-sitter、權限 → 全本地 |
| 雲端執行層 | 模型推理、Embedding、搜索、壓縮 → 走 API |
| 邊界優化 | 哪些雲端推理可下移本地？哪些緩存可預計算？ |
| Hook 位置 | Hook 全在本地 = 零雲端成本 = 核心擴展位 |
