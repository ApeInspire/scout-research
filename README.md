# Scout Research

独立前沿科技研究机构。一人研究机构。研究 AI 时代的技术趋势、产业变革、一人×AI 模式。

**研究者：** [Ape](https://github.com/ApeInspire)
**方法：** 研究 → 分析 → 发布 → 公开验证 → 持续校准

---

## 目录

```
research/    — 深度研究报告（月度）
notes/       — 研究笔记（周度）  
predictions/ — 可验证预测追踪
data/        — 准确率统计与方法论
```

## 研究领域

- AI 技术演进（模型、架构、协议、工具链）
- 软件范式转移（SaaS → AI-Native）
- 一人×AI 公司模式
- AI 产业格局与职业重塑
- 香港/亚洲 AI 生态

## 原则

- 所有预测可验证（时间窗口 + 验证指标）
- 所有判断有依据（信號来源 + 推理链）
- 准确率公开追踪
- 错误公开承认，方法论持续迭代
